1. Bid on Bugs

Find a frustrating bug? Need an important feature? Chip some cash in towards the solution. Badoss will add your bid to others' and alert you when the work is done.

2. Bid on Fixing

How much would it take you to fix it yourself? Before you code around it, tell Badoss how much you would want to fix the issue yourself. Badoss will alert you if bidders are willing to pay it.

3. (Non-)Profit

More open-source software for everyone! Badoss is run by a non-profit to support open-source software.
